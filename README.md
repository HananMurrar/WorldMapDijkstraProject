### Word map project
This project allows users to find and visualize the shortest route between countries using a world map

#### Steps:
- Click the load button to read the data file
- You can select the source and target locations either by clicking on the points on the map or by choosing from the combo box
- If a path exists from the source to the target country, it will be drawn on the map
- If no path exists from the source to the target country, an alert message will appear
